# 使い方

Markdownをwikiの記法に直します。

インストール
```zsh
npm i
```

Markdownを.txt形式で出力
```zsh
npm run t
```

出力結果の文字数カウント
```zsh
npm run c
```

出力とカウントを同時に
```zsh
npm run tc
```

MarkdownをHTMLで出力
```zsh
npm run h
```

# 対応済み

- 見出し
- リスト
- 段落内改行
- リンク
- 整形済みbox
- 打ち消し線
- 太字

# 非対応

- 表
- 定義
- 太字と打ち消し専以外の文字装飾

# 注意

editorconfigをいじると壊れる可能性があります。
